# 🐍 Processamento e Automação de Dados

## 📝 Descrição do Projeto
Este repositório contém um conjunto de ferramentas lógicas desenvolvidas em Python para a resolução de problemas cotidianos e automação de fluxos de dados. O projeto foi estruturado para demonstrar a aplicação de estruturas de controle (loops, condicionais), manipulação de tipos primitivos e interação com o usuário via console.
O notebook principal (Untitled1.ipynb) integra quatro módulos funcionais independentes, cada um focado em uma regra de negócio específica: processamento financeiro, monitoramento ambiental e gestão acadêmica.

## 🛠️ Funcionalidades Principais
1. 💰 Processamento de Vendas
Um sistema de checkout que calcula o total de uma compra com base na quantidade de itens e aplica descontos progressivos:
Desconto de 10% para compras acima de R$ 500,00.
Desconto de 5% para compras acima de R$ 200,00.
Validação de entrada para evitar preços ou quantidades negativas.

2. 🌤️ Analisador de Clima Semanal
Ferramenta de monitoramento que processa dados térmicos de uma semana (7 dias):
Calcula a média de temperatura semanal.
Contabiliza dias com calor excessivo (acima de 35°C).
Emite Alertas de Condições Extremas para temperaturas acima de 45°C ou abaixo de -5°C.

3. 🎓 Sistema de Notas Acadêmicas
Interface para gestão de turmas que automatiza o cálculo de desempenho:
Processamento de médias aritméticas individuais.
Classificação automática: Aprovado (>= 7.0), Recuperação (>= 5.0) ou Reprovado.

4. 📈 Simulador de Poupança Progressiva
Calculadora financeira para projeção de investimentos a longo prazo:
Considera um saldo inicial e taxas de juros mensais variáveis.
Permite aportes mensais dinâmicos.
Notifica o usuário ao atingir metas de patrimônio (ex: R$ 10.000,00).

## 🚀 Tecnologias Utilizadas
Linguagem: Python 3.x
Ambiente de Desenvolvimento: Jupyter Notebook / Google Colab
Paradigma: Programação Estruturada

## 🔧 Como Executar
1. Abra o arquivo Untitled1.ipynb.
2. Execute as células de código sequencialmente.
3. Siga as instruções no console para inserir os dados solicitados por cada função.
